There are four datasets and their columns listed below:

1. Album (TSV file - tab separated)
album_id
album_name
album_release_date
album_artist_id

2. Artist (TSV)
artist_id
artist_name
artist_genre
artist_popularity
artist_followers

3. Track (TSV)
track_id
track_name
track_artist_id
track_album_id
track_popularity
track_explicit

4. Audio Features (CSV file - comma separated)
track_id
track_duration (in milliseconds)
danceability
energy
loudness
speechiness
acousticness
instrumentalness
liveness
valence
tempo
